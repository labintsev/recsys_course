# Movie Lens small

## Подробное описание данных

Этот набор данных (ml‑latest‑small) описывает действия пользователей сервиса рекомендаций фильмов [MovieLens](http://movielens.org) — выставление оценок по 5‑балльной шкале и добавление текстовых тегов. Он содержит **100 836 оценок** и **3 683 тега** для **9 742 фильмов**. Данные были созданы **610 пользователями** в период с 29 марта 1996 г. по 24 сентября 2018 г. Набор данных сформирован 26 сентября 2018 г.

Пользователи выбирались для включения в набор случайным образом. Все отобранные пользователи оценили не менее 20 фильмов. Демографическая информация не включена. Каждый пользователь представлен идентификатором (id), иных данных не предоставлено.

Данные содержатся в файлах `links.csv`, `movies.csv`, `ratings.csv` и `tags.csv`. Ниже приведены подробности о содержимом и использовании этих файлов.

Это набор данных для *разработки*. Он может изменяться со временем и не подходит для публикации результатов совместных исследований. Если вам нужен набор для бенчмаркинга, ознакомьтесь с доступными *benchmark*-наборами данных.

Этот и другие наборы данных GroupLens доступны для скачивания на <http://grouplens.org/datasets/>.

### Содержимое и использование файлов

#### Форматирование и кодировка

Файлы набора данных представлены в формате [CSV (значения, разделённые запятыми)](http://en.wikipedia.org/wiki/Comma-separated_values) с одной строкой заголовка. Столбцы, содержащие запятые (`,`), экранируются двойными кавычками (`"`). Файлы закодированы в UTF‑8. Если акцентированные символы в названиях фильмов или значениях тегов (например, *Misérables, Les (1995)*) отображаются некорректно, убедитесь, что любая программа, читающая данные (текстовый редактор, терминал или скрипт), настроена на UTF‑8.

#### Идентификаторы пользователей (User Ids)

Пользователи MovieLens выбирались для включения случайным образом. Их идентификаторы (id) анонимизированы. Идентификаторы пользователей согласованы между файлами `ratings.csv` и `tags.csv` (т. е. один и тот же id относится к одному и тому же пользователю в обоих файлах).

#### Идентификаторы фильмов (Movie Ids)

В набор данных включены только фильмы, имеющие хотя бы одну оценку или тег. Эти идентификаторы фильмов согласованы с теми, что используются на веб‑сайте MovieLens (например, id `1` соответствует URL <https://movielens.org/movies/1>). Идентификаторы фильмов согласованы между файлами `ratings.csv`, `tags.csv`, `movies.csv` и `links.csv` (т. е. один и тот же id относится к одному и тому же фильму во всех четырёх файлах данных).

#### Структура файла с данными об оценках (ratings.csv)

Все оценки содержатся в файле `ratings.csv`. Каждая строка этого файла после строки заголовка представляет одну оценку одного фильма одним пользователем и имеет следующий формат:

```
userId,movieId,rating,timestamp
```

Строки в этом файле упорядочены сначала по `userId`, затем — внутри пользователя — по `movieId`.

Оценки выставляются по 5‑балльной шкале с шагом в полбалла (от 0,5 до 5,0 звёзд).

Временные метки (timestamp) представляют количество секунд, прошедших с полуночи по Всемирному координированному времени (UTC) 1 января 1970 г.

#### Структура файла с данными о тегах (tags.csv)

Все теги содержатся в файле `tags.csv`. Каждая строка этого файла после строки заголовка представляет один тег, применённый к одному фильму одним пользователем, и имеет следующий формат:

```
userId,movieId,tag,timestamp
```

Строки в этом файле упорядочены сначала по `userId`, затем — внутри пользователя — по `movieId`.

Теги — это метаданные о фильмах, созданные пользователями. Каждый тег обычно представляет собой одно слово или короткую фразу. Значение, смысл и цель конкретного тега определяются каждым пользователем индивидуально.

Временные метки (timestamp) представляют количество секунд, прошедших с полуночи по Всемирному координированному времени (UTC) 1 января 1970 г.

#### Структура файла с данными о фильмах (movies.csv)

Информация о фильмах содержится в файле `movies.csv`. Каждая строка этого файла после строки заголовка представляет один фильм и имеет следующий формат:


```
movieId,title,genres
```

Названия фильмов вводятся вручную или импортируются с <https://www.themoviedb.org/> и включают год выпуска в скобках. В этих названиях могут встречаться ошибки и несоответствия.

Жанры представлены в виде списка, разделённого вертикальной чертой, и выбираются из следующего перечня:

* Action;
* Adventure;
* Animation;
* Children's;
* Comedy;
* Crime;
* Documentary;
* Drama;
* Fantasy;
* Film‑Noir;
* Horror;
* Musical;
* Mystery;
* Romance;
* Sci‑Fi;
* Thriller;
* War;
* Western;
* (no genres listed).

#### Структура файла со ссылками (links.csv)

Идентификаторы, которые можно использовать для связи с другими источниками данных о фильмах, содержатся в файле `links.csv`. Каждая строка этого файла после строки заголовка представляет один фильм и имеет следующий формат:

```
movieId,imdbId,tmdbId
```

* `movieId` — идентификатор фильмов, используемый на <https://movielens.org>. Например, для фильма *Toy Story* ссылка выглядит так: <https://movielens.org/movies/1>.
* `imdbId` — идентификатор фильмов, используемый на <http://www.imdb.com>. Например, для фильма *Toy Story* ссылка выглядит так: <http://www.imdb.com/title/tt0114709/>.
* `tmdbId` — идентификатор фильмов, используемый на <https://www.themoviedb.org>. Например, для фильма *Toy Story* ссылка выглядит так: <https://www.themoviedb.org/movie/862>.

Использование перечисленных выше ресурсов регулируется условиями каждого поставщика.
